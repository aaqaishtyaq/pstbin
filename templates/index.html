NAME
    pstbin: A simple command line pastebin.

SYNOPSIS
    <command> | curl -F 'paste=<-' {{.URL}}

DESCRIPTION
    use this form to paste from a browser
    add ?hl=true to resulting url for highlight.js syntax highlighting via cdnjs

EXAMPLES
    ~$ cat file.txt | curl -F 'paste=<-' {{.URL}}
       {{.URL}}aXZI
    ~$ firefox {{.URL}}aXZI
